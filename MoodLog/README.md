# MoodLog

A macOS app to log your mood and emotional state throughout the day.

## Features

- Record your mood on a 5-level scale (Very Bad → Great) with an optional note
- Filter entries by mood level using the sidebar
- Grouped by day for easy review
- Data persisted to `~/Library/Application Support/MoodLog/entries.json`

## How to build

1. Open Xcode → **File → New → Project → macOS → App**
2. Set language to **Swift**, interface to **SwiftUI**
3. Delete the default `ContentView.swift` generated by Xcode
4. Drag all `.swift` files from this folder into the Xcode project (ensure "Copy items if needed" is checked)
5. Set **Deployment Target** to macOS 13.0 or later
6. Build and run (⌘R)

## Requirements

- macOS 13.0+
- Xcode 14+
