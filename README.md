# Mac Apps Factory

A collection of macOS SwiftUI apps focused on **行動記録 (activity & behavior recording)**, automatically generated by a Claude cloud agent on a recurring schedule.

## What's here

Each folder contains a standalone SwiftUI macOS app. To build any app:
1. Open Xcode → File → New → Project → macOS → App
2. Add the `.swift` files from the app's folder to the project
3. Build & Run (⌘R)
